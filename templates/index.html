What is a Storage Virtual Machine (SVM)?

A Storage Virtual Machine (SVM, also called a vserver) is a logical container inside your NetApp ONTAP cluster that holds volumes, network interfaces, and protocols. Think of it like a "virtual NAS filer" — your Domino NetApp volume lives inside one.

The dropdown is auto-loaded from your ONTAP cluster — you do not need to know the SVM name in advance. Simply pick the one that hosts your Domino data. If you are unsure which SVM to choose, ask your NetApp or infrastructure administrator.

How to verify the correct SVM (optional)
  1. Open the NetApp ONTAP System Manager web UI (your admin can provide the URL — typically https://<cluster-ip>).
  2. Navigate to Storage → Storage VMs.
  3. Look for the SVM that has a CIFS/SMB protocol enabled and whose volumes contain your Domino project data.

If you only see one SVM in the dropdown, that is almost certainly the correct one.
